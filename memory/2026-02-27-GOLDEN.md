### 🔑 Key Learnings:

1. **Model Syntax:** Use `moonshot/kimi-k2.5` (with dot), NOT `kimi-k2p5` (with p) or `kimi-compact`
2. **Fallbacks Array:** Use `"fallbacks": [...]` (plural), NOT `"fallback"` (singular)
3. **Sub-agent Spawning:** Use `"agents.list"` with `"subagents.allowAgents": ["*"]` - this WORKS
4. **AVOID:** `"tools.agentToAgent.enabled": true` - this BREAKS sub-agent spawning (known bug #5813)
5. **Token Sync:** If gateway token mismatch occurs, run `openclaw gateway install --force` then restart
6. **Security:** ALWAYS rotate tokens after any exposure. Use `allowAgents: ["*"]` for full sub-agent access.
